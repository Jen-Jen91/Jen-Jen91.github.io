Recent Work

Pacman Invaders

A 2D fixed shooter game built in Unity and C#. A mash-up of Pacman and Space Invaders - players use Pacman to shoot incoming ghosts and collect fruit for points.

To Do

A 'To Do' mobile app built with React Native and JavaScript. It uses AsyncStorage to persist the task data and React Navigation to move between the task lists.

Block Breaker

A 2D game made with Unity and C#, making use of object colliders and gravity. The player bounces the ball off a paddle to break the floating blocks.

Baking Recipe Finder

A full-stack web app which helps users dicover recipes based on the ingredients they have at home. A small API was built in Java and Spring, and the front end was create in React.

Pokemon Selector

A React web app that makes two API requests to get information on the original 151 Pokemon. Users select from a dropdown to find out more about their favourite.

Game Shop Inventory Tracker

An inventory tracker web app for game shop owners, built in Ruby. It uses a PostgreSQL database to store video game information which users can view, update, add to, and delete.

Flight Information

A front-end web app created with React and TypeScript. Users can view all necessary information about flight arrivals and departures at Edinburgh Airport.

The Lost Prince

A text-adventure browser game made in JavaScript, with in-game music and pixel art. The player makes decisions and uses dice roll mechanics to combat enemies.
